# 本ツールの概要
GeminiとのQAを抜き出して、ウェブページを生成・ブラウザの印刷機能でPDF化するための拡張機能です。

# 使用方法 (2026-05-10現在)
1. このツールをローカルに取得
2. chrome://extensions/ をブラウザのURLに入力、拡張機能ページを開く
3. "パッケージ化されていない拡張機能を読み込む"のボタンをクリック
4. 本ツールのフォルダを開く
5. PDF化したいGeminiとのチャットページを開く(HOMEボタンで、先頭から末尾まで読み込む)
6. ブラウザ右上の、拡張機能ボタンをクリックして、"Gemini Answer Exporter"をクリック
7. ウェブページが開いたら、ブラウザの"印刷"ボタンでPDFを生成

# 仕様変更によるエラーが出た場合
生成AIにcontent.jsの中身を確認してもらい、修正してください。
